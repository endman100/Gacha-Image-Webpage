# Gacha Image Webpage

This project is a front-end webpage for browsing LoRA data, editing scene prompts, and generating images.

## Open via GitHub Pages

https://endman100.github.io/Gacha-Image-Webpage/

## How to Use

1. Open the link above and wait for the page to finish loading.
2. Pick a LoRA from the list.
3. Enter or edit the scene description in the prompt area.
4. Click generate when ready.
5. Preview the result and adjust if needed, then try again.
