# Gacha Image Webpage

This project is a front-end webpage for browsing LoRA data, editing scene prompts, and generating images.

## Open via GitHub Pages

https://endman100.github.io/Gacha-Image-Webpage/

## How to Use

Step 1: Access the Webpage

Open your browser and navigate to the following URL:
https://endman100.github.io/Gacha-Image-Webpage/

Step 2: Unlock Access (Required for First-Time Use)

In the Password input field, enter the correct password (provided by the project owner).
Press the Enter key, or click the Search LoRA Files button.
The password will be automatically saved to the browser's Cookie (valid for 30 days) and will be loaded automatically on subsequent visits.


Step 3: Search and Load LoRA Models

Click the Search LoRA Files button.
Wait for the loading process to complete. The interface will display the list of LoRA models in the form of a card gallery.

Step 4: Enter the Model Detail Page

Click on any LoRA card to access the detailed view of that model.
This page includes the following sections:
Model Information
General Character Scene Table
Character-Specific Scene Table
Generation Control Panel


Step 5: Manage and Edit Scenes

Add Scenes:
Click Add Row to insert a new blank row.
Alternatively, click Add Row by AI to allow the AI to automatically generate scene designs and prompts.

Edit Prompts:
Enter your description in the Scene Design column.
Click the Convert to Prompt button to have the AI convert the description into a complete prompt.
Check the Keep Outfit option to retain the character's original clothing.

Save Changes:
Once editing is complete, click the Save button to store the data on the backend.


Step 6: Generate Images

After verifying that the prompts are accurate, click the Generate Gacha button or the Runpod generation button.
Wait for the generation process to complete (the duration depends on GPU load).
The generated results will appear in the gallery section below. You may click on any image to enlarge it for preview or sort the results as needed.
