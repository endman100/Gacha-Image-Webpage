# Gacha Image Webpage

This project is a front-end webpage for browsing LoRA data, editing scene prompts, and generating images.

## Open via GitHub Pages

https://endman100.github.io/Gacha-Image-Webpage/

## How to Use

1. Open the link above and wait for the page to finish loading.
2. Pick a LoRA from the list.
3. Enter or edit the scene description in the prompt area.
4. Click generate when ready.
5. Preview the result and adjust if needed, then try again.

## Button Reference

### Main View

- Search LoRA Files: Load and display available LoRA items.
- ‚Üê Back: Return to the main view from the detail view.

### Detail View

- Save (General Character Scenes): Save the General Character Scenes table.
- Add Row (General Character Scenes): Add a new row to the General Character Scenes table.
- Add Row By AI (General Character Scenes): Add a row and auto-generate the scene and prompt.
- Save (Character-Specific Scenes): Save the Character-Specific Scenes table.
- Add Row (Character-Specific Scenes): Add a new row to the Character-Specific Scenes table.
- Add Row By AI (Character-Specific Scenes): Add a row and auto-generate the scene and prompt.
- Generate Gacha: Start generating images with the current settings.
- Sort: Ascending/Descending: Toggle the sort order of the result images.
- Remove (table row): Delete the selected scene row.
- Convert to Prompt (table row): Convert the scene description into a prompt.
